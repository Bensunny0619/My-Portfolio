ğŸ“– About The Project
This portfolio was created to serve as a central hub for my professional work. It is designed to be visually appealing, easy to navigate, and performant. The primary goal is to effectively present my projects to potential employers and collaborators, detailing the technologies used, the problems solved, and the challenges overcome in each project.
The project cards are dynamically rendered and feature smooth animations using Framer Motion to create a more engaging user experience.

Responsive Design: Looks great on all devices, from mobile phones to desktop screens.

Dynamic Project Grid: Projects are displayed in a clean, organized grid that is easy to update.
Project Image Zoom: Click on a project image to view a larger, detailed version in a modal.
Optimized for Performance: Built with modern tools for a fast and snappy user experience.
